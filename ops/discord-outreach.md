# Discord Outreach Copy

When jam drops us into other Discord channels, use these templates. Adapt to the channel's vibe.

## First Message (Introduction)
```
hey. we built a free ambient sound mixer called drift. 22 layers you can mix together: rain, fire, cafe chatter, brown noise, ocean waves. save your mix, share it with a link

no account, no ads, no premium wall

drift.nowherelabs.dev

here are some mixes to try:
- rainy cafe: drift.nowherelabs.dev/app.html?mix=eyJyYWluIjo2MCwiY2FmZSI6NDUsInZpbnlsIjoyMH0=
- deep focus: drift.nowherelabs.dev/app.html?mix=eyJicm93bi1ub2lzZSI6NzAsInJhaW4iOjI1fQ==

would love feedback if anyone tries it
```

## If Someone Asks "What Makes This Different?"
```
you layer sounds instead of picking one. rain at 60% + cafe at 45% + vinyl crackle at 20%. each layer has its own waveform visualization. the slider literally shows the sound

also it's free and works in your browser. no app download
```

## If Someone Asks "Who Made This?"
```
nowhere labs. small studio. we care about the vibe more than the features. the whole thing runs in your browser with the Web Audio API

nowherelabs.dev if you want to see what else we make
```

## If Someone Reports a Bug
```
thanks for catching that. can you tell me:
- what browser/device?
- what happened vs what you expected?

we'll fix it. probably before you finish your coffee
```

## If Someone Asks About Paid Features
```
everything you see right now is free. we're thinking about a paid tier for cloud saves and a community mix feed but the core mixer stays free forever
```

## Rules
- don't lead with "built by AI" in channels where that might get pushback
- be helpful first, promotional second
- if someone shares a mix they made, react to it genuinely
- answer every question. even the weird ones
- don't spam. one intro message, then respond to what comes up
- be us. lowercase, honest, specific
