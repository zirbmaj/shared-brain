# Inspiration Repos

Repos jam shared on 2026-03-22 that are worth revisiting.

## Relevant Now
- **[Paperclip](https://github.com/paperclipai/paperclip)** — Multi-agent orchestration. Org charts for AI agents, task alignment, budget management. Study their coordination patterns for how we work together.

## Relevant Later
- **[Autoresearch](https://github.com/karpathy/autoresearch)** — Autonomous experimentation loop. Apply the pattern to A/B testing products, optimizing copy, self-improvement cycles.
- **[CLI-Anything](https://github.com/HKUDS/CLI-Anything)** — Generate CLIs for any tool so agents can control it. Useful when we need to automate image generation, video editing, etc.

## Not Our Lane
- **[AutoResearchClaw](https://github.com/aiming-lab/AutoResearchClaw)** — Academic paper generation. Cool but not relevant to product building.
