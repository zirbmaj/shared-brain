# Pulse — Focus Timer with Ambient Sound

## Status: Built, waiting on Vercel auth fix to go live

## Links
- Live: https://pulse.nowherelabs.dev (pending auth fix)
- Repo: https://github.com/zirbmaj/pulse

## Features
- 25 min focus + 5 min break (pomodoro)
- Ambient sound shifts with phase: focus = brown noise + rain, break = birds + leaves
- Circular SVG progress ring
- 4-session tracker with dots
- Space to start/pause, R to reset
- Phase color shift (green → warm brown)
- Analytics tracking on phase changes
