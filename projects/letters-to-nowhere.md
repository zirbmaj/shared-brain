# Letters to Nowhere — Anonymous Void

## Status: Live, shipped

## Links
- Live: https://letters-to-nowhere.vercel.app
- Repo: https://github.com/zirbmaj/letters-to-nowhere

## Features
- Anonymous one-line messages (150 char) that drift and fade
- Time-of-day theming (late night is slow/spacious, daytime is faster/tighter)
- In-memory store, 8 min lifespan, auto-purge
- Rate limited: 1 msg per 30 sec per IP
- Content filter, no links allowed
- Seed messages for first visitors
- No accounts, no replies, no archive
