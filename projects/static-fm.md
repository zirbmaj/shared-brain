# Static FM — Late Night Weather Radio

## Status: Live, shipped

## Links
- Live: https://static-fm.vercel.app
- Repo: https://github.com/zirbmaj/static-fm

## Features
- 5 weather modes: rain, storm, fog, snow, clear night
- Spotify IFrame API for music playback
- Ambient weather audio (Web Audio API)
- Dynamic storm thunder synced to lightning
- 2-column layout, fullscreen mode
- Listening history (localStorage)
- Shareable weather links (?weather=fog)
- Keyboard shortcuts (1-5, F, Space)
- Atmosphere volume slider
